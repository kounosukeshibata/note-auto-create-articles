# プロダクト概要

## ビジョン
「収益をデザインする」
AI（Gemini & Imagen）とアフィリエイトエコシステムを統合し、モバイルから数タップで高品質な収益化記事を生成・投稿する。

## 目的
収益化記事の生成から投稿までをモバイルから数タップで完結させる。

## 差別化ポイント
1. **DDDによる堅牢性:** 複雑なアフィリエイトロジック（商品検索、リンク置換）をドメイン層に隔離し、高いメンテナンス性を確保。
2. **AIマルチモーダル:** 本文（Gemini）とアイキャッチ（Imagen）を同時に最適化生成。自動品質レビューによる高品質な記事生成を行う．
3. **モバイルUX:** TanStack Router によるストレスのない操作感。

## 主要機能
1. **AI自動品質レビューシステム:**
   - 生成コンテンツのSEO適合性（キーワード密度・構成）、薬機法/ASP規約の遵守、アフィリエイトリンクの配置最適性をAIが自動診断し、フィードバック・修正を行う機能。
2. **モバイル最適化UI:**
   - モバイルからの操作に特化した、高速な記事入力・管理フォーム。
   - モバイル画面での入力項目は，売れる記事を自動生成するために必要な項目を網羅．
3. **AIマルチモーダル生成:**
   - 記事本文生成: Gemini 2.5 Flash（文字数目安: 1000/2000/3000/4000字から選択可能）
   - 画像生成: Imagen 3（記事テーマに合わせたアイキャッチの自動生成）
4. **アフィリエイト自動化パイプライン:**
   - キーワード抽出（LLMによるSEO最適化）
   - 商品検索（外部アフィリエイトAPI連携）
   - リンク置換（生成テキストへの追跡用リンクの自動挿入）
   - 楽天とAmazonのアフィリエイトは，より還元率が高い方を採用

## MVPスコープ
1. モバイル画面はログイン画面があり，新規登録機能と登録済みアカウントのログイン機能をどちらももつ
2. モバイルWebでの記事生成・画像生成に必要なプロンプト項目の入力が可能
3. AIによる記事＆アイキャッチ画像&アフィリエイトリンクの生成
4. 生成物のをモバイル画面でプレビュー可能
5. モバイル画面で保存ボタンを押下すると，データベースに保存するとともに，noteに一時保存する（最終的な投稿は手動で行う）

## 成功指標
- 記事生成から note 投稿までのリードタイム短縮
- 記事のSEOキーワード適合率向上
